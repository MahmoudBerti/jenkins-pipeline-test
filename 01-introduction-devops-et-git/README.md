# Module 01 — Introduction au DevOps et Git de base

Premier module du cours **Développement et déploiement de solutions de données** (420-D30-BB).
Il pose les fondations : la culture DevOps, puis la maîtrise de Git en local et à distance.

## Objectifs

À la fin de ce module, vous serez capable de :

- Expliquer ce qu'est la **culture DevOps** (collaboration, automatisation, rétroaction).
- **Installer et configurer** Git sur votre poste.
- Créer un **dépôt local**, suivre des fichiers et rédiger de bons **commits**.
- Travailler avec des **branches**, fusionner et consulter l'**historique**.
- Connecter un **dépôt distant** (GitHub) et collaborer via `push` / `pull` / `clone`.
